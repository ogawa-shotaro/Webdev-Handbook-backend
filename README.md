# Web開発ハンドブック（バックエンド）

個人学習として制作した **Todoアプリ開発**を通じて身につけたバックエンドの必須知識をまとめた学習ノートです。
エンジニアによる指導・レビューを受けながら整理した内容を、参照しやすいハンドブック形式にまとめました。
`index.html` を開くだけでローカル環境でも動作します。

---

## 概要

| 項目 | 内容 |
|------|------|
| 対象 | バックエンド開発を学ぶエンジニア・学習者 |
| 形式 | シングルページHTML（サイドバーナビ付き） |

---

## 収録トピック

### 基礎
- **バックエンド基本（Express で理解する）**
  サーバーの役割・HTTPリクエスト/レスポンスの流れ・ルーティングの基本を Express を題材に解説

- **Node.js**
  実行環境の概要・非同期処理・モジュールシステム・よく使うコアモジュール

- **npm**
  パッケージ管理ツールの使い方・`package.json` の読み方・依存関係の管理

---

### Laravel（PHP フレームワーク）
- **Laravel**
  フレームワーク全体の概念・MVC アーキテクチャ・ディレクトリ構成

- **Composer**
  PHP のパッケージマネージャ。インストール・`composer.json` の管理方法

- **ルーティング**
  `web.php` / `api.php` によるルート定義・名前付きルート・グループ化

- **ビュー（View / Blade）**
  Blade テンプレートエンジンの構文・レイアウト継承・コンポーネント

- **コントローラー**
  責務の分離・リソースコントローラー・フォームリクエスト

- **モデル（Eloquent ORM）**
  モデル定義・リレーション（1対1・1対多・多対多）・クエリビルダ

- **ミドルウェア**
  リクエスト前後の処理・認証ガード・グローバル vs ルート単位の適用

- **マイグレーション**
  テーブル定義のバージョン管理・`up` / `down` メソッド・よく使うカラム型

## 使い方

```bash
# リポジトリをクローン
git clone https://github.com/<your-username>/Webdev-Handbook-backend.git

# index.html をブラウザで開く
open index.html   # macOS
# または
start index.html  # Windows
```

サーバー不要でブラウザのみで動作します。
